# YouTube Time Limiter

A Chrome extension that helps reduce YouTube overconsumption by tracking daily watch time and enforcing a time limit.

## Features (MVP)

- Track watch time on YouTube videos
- Count time only when video is actively playing
- Daily usage tracking
- Configurable daily limit
- Block playback when limit is reached

## Tech Stack

- JavaScript
- Chrome Extensions API
- HTML / CSS
